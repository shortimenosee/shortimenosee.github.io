regex正则匹配应用

  • 需要提取json文件中特定键值对，之前使用jar包将json文件转为json object， 今天尝试一下利用正则匹配查找

工具

java.util.regex

匹配规则

  1. \  将下一字符标记为特殊字符
  2. ^  从待匹配串第一个字符开始匹配
  3. $  从待匹配串末尾进行匹配

在线正则表达测试

https://tool.oschina.net/regex

示例

String text = "Phone number: 123-123-213"; //待匹配
String regex = "(\\d{3}-(\\d{3}))"; //模式串&正则表达式
Pattern pattern = Pattern.compile(regex);//编译模式串
Matcher matcher = pattern.matcher(text); //建立一个matcher
matcher.find(); //寻找相匹配的子串
matcher.match(); //查看串整体是否与模式串匹配成功
matcher.group();//根据分组来查看匹配成功的子串
